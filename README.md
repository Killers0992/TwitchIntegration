# TwitchIntegration
Twitch bot which allows you to manipulate vrchat avatar thru OSC.

Currently implemented events:
- Commands
- OnReSubscribe / OnSubscribe
- OnBanned / OnTimedout
- OnUseReward
- OnReceiveBits
- OnBeingHosted
- OnFollow
